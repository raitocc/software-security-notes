# 软件安全复习资料

基于课程 PPT 整理的复习手册和配套习题，覆盖 Lecture01-Lecture09 全部内容。

## 目录结构

```
├── 软件安全复习手册/   # 9讲复习手册
└── 习题与答案/         # 各讲习题、答案及综合练习题
```

## 内容概览

| 讲次 | 主题 |
|---|---|
| Lecture01 | 课程概要和基本概念 |
| Lecture02 | 堆栈基础和汇编语言 |
| Lecture03 | 软件调试基础 |
| Lecture04 | 缓冲区溢出漏洞 |
| Lecture05 | 其他常见漏洞 |
| Lecture06 | 漏洞利用基础 |
| Lecture07 | 漏洞利用防护与漏洞挖掘 |
| Lecture08 | Web 安全 |
| Lecture09 | 渗透测试基础及实践 |

## 使用建议

1. 按顺序阅读 `软件安全复习手册/` 中的各讲手册
2. 每讲读完后做 `习题与答案/` 中对应的习题，再核对答案
3. 全部完成后做 `习题与答案/综合练习题.md` 进行融会贯通练习

## 习题设计

- 三层递进：基础查表 → 理解应用 → 综合融会
- 干扰项经过优化，避免一眼排除
- 答案解析详细说明每个错误选项为什么错
