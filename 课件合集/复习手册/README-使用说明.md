# 软件安全复习手册使用说明

## 目录结构

```
复习手册/
├── 手册/           # 9讲复习手册（Lecture01-09）
├── 习题与答案/     # 各讲习题、答案及综合练习题
└── README-使用说明.md
```

## 推荐阅读顺序

1. 按课程顺序阅读 `手册/` 目录下的 `Lecture01` 到 `Lecture09` 复习手册。
2. 每讲先读"本讲主线"，再读各小节，最后用"复习自查"检查掌握情况。
3. 有配套习题的讲次，建议先做 `习题与答案/LectureXX-...-习题.md`，再看 `习题与答案/LectureXX-...-习题答案.md`。
4. 完成各讲习题后，做 `习题与答案/综合练习题.md` 进行融会贯通练习。
5. 原始抽取文件和逐页文字版已移到隔壁的 `课件处理中间文件`，需要人工复核 PDF 时再看。

## 合并策略

- Lecture01-Lecture06：每讲单独处理。
- Lecture07：合并 `lecture07part1-漏洞利用防护.pdf` 和 `lecture07part2-漏洞挖掘.pdf`。
- Lecture08：合并 `lecture08part1-Web安全.pdf` 到 `lecture08part4-Web安全.pdf`。
- Lecture09：使用 `reading-material-渗透测试.pdf`，作为渗透测试基础及实践。
- `漏洞利用及渗透测试基础第2版.pdf` 是教材书本，未并入本轮课件手册。

## 文件说明

| 目录/文件 | 用途 |
|---|---|
| `手册/LectureXX-...-复习手册.md` | 单讲复习手册，按课程内容重组并补足解释 |
| `习题与答案/LectureXX-...-习题.md` | 单讲配套练习，按基础查表、理解应用、综合融会分层 |
| `习题与答案/LectureXX-...-习题答案.md` | 单讲配套答案和解析，和习题分离 |
| `习题与答案/综合练习题.md` | 跨讲综合练习，侧重融会贯通和场景分析 |
| `习题与答案/综合练习题答案.md` | 综合练习题答案 |

## 覆盖说明

复习手册负责"适合直接复习"，对 PPT 内容做了重组、解释和对比。中间文件目录保留逐页抽取结果，方便以后核对来源。

图像页的处理方式：

- 纯装饰或过渡页不单独扩写。
- 对学习有意义的截图或图表，在手册中转写为概念解释。
- 代码演示和攻击工具截图只保留原理与授权实验语境，不整理成无授权操作手册。
